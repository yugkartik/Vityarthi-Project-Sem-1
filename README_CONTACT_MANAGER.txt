A simple Contact Management System built using Python.
This program allows users to add, view, search, update, and delete contacts and stores all information permanently in a text file.

-----------------------------------------------------

FEATURES
- Add new contacts
- View all saved contacts
- Search contacts by name
- Update existing contact information
- Delete contacts
- Permanent data storage using contacts.txt
- Menu-driven interface
- Modular Python code

-----------------------------------------------------

HOW TO RUN
1. Install Python 3
2. Save the file as: contact_list_manager.py
3. Run in terminal:

python contact_list_manager.py

-----------------------------------------------------

DATA FORMAT (contacts.txt)

name,phone,email

Example:
John,9876543210,john@gmail.com
